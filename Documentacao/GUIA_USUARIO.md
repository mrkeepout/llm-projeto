# Guia do Usuário

## Instalação

1. Clone o repositório
2. Instale dependências: `pip install -r requirements.txt`

## Uso

### Executar Backend

```bash
cd Backend
python main.py
```

### Executar Frontend

Abra `Frontend/index.html` no navegador.

## Otimização Manual

Para ver a demonstração do Gradient Descent implementado manualmente:

```bash
python Análise/otimizacao_manual.py
```

Isso gerará um gráfico comparativo `comparacao_otimizacao.png`.
